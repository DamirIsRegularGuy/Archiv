     id SERIAL PRIMARY KEY,
     name VARCHAR(50),
     age INT
);
тут я создаю таблицу с тримя полям: id, name, age
где id самозаполняющегося на единцу, на что указывает SERIAL, а PRIMARY KEY 
указывает на укальность данного поля, а с name -> строковой тип, длиной 50 символов,
age -> числовой
